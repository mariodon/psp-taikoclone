OldSchool Library (OSLib) MOD
-----------------------------
Source code online: http://trac2.assembla.com/oslibmod/browser/trunk
	    svn   : http://svn2.assembla.com/svn/oslibmod/trunk


This is a modded versione of OSLib 2.10
The original OSLib by Brunni can be found here:
http://brunni.palib.info/new/index.php?page=pspsoft_oslib

Check the changelog to see changes from 2.10
Check also the samples directory


DOCUMENTATION
-------------
You can find the documentation in the Doc directory, or consult it online here:
http://www.sakya.it/OSLib_MOD/doc/html/


SUPPORT
-------
You can report bugs and talk about OSLib MOD here:
http://www.sakya.it/forums/viewforum.php?f=7


INSTALL
-------
To install OSLib MOD copy
libosl.a in $PSPSDK/lib/

copy the oslib directory (containing header files) in $PSPSDK/include/
